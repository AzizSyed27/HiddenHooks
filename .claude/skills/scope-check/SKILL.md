---
name: scope-check
description: Review what is currently being built against Phase 1's stated deliverables and flag any scope creep.
allowed-tools: Read, Glob
---

Review the current state of the project against Phase 1's stated deliverables:

1. Read CLAUDE.md, any planning docs, README, or specs to find the Phase 1 definition of done
2. Survey what has actually been built so far (files changed, features added, endpoints created, etc.)
3. Compare the two and flag:
   - **In scope** — work that clearly maps to Phase 1 deliverables
   - **Scope creep** — work that goes beyond Phase 1 (even if useful)
   - **Missing** — Phase 1 items not yet addressed
4. Give a plain-English summary and a recommended next action

Be direct. The goal is an honest audit, not reassurance.
